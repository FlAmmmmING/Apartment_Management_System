# Apartment_Management_System
国内大学教育多为住宿制，需要有工作人员不停地管理学生公寓的秩序。而21世纪，随着国内高等教育大量扩建扩招，每年大学生的数量都不停上涨，大大加重了管理的难度。故而，一所住宿制的大学，一定需要有一个功能齐全、操作简便的学生公寓管理系统，才能大大减少管理的压力以及增强学生公寓的安全保障。
为了解决这个问题，我们在设计开发这个系统的基础上，分析了学生公寓管理可能用到的扩展需求，比如在我们的系统中，添加了外来访问者管理系统，增强了学生公寓的安全性，同时增加了一个公寓学生会，提供这个平台让同学能够更加方便地组织管理各项活动。
本系统的实现能够提高学生公寓的管理效率，实用性强，且运行稳定，使用方便，内置许多扩展功能模块，可以满足管理者对学生公寓的管理需求。
